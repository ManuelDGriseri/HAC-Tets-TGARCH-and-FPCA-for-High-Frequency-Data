# HAC-tets-TGARCH-and-FPCA-for-high-frequency-data
Financial econometrics project divided into three parts.

-The first part focuses on conducting appropriate tests for autocorrelation on the difference in volatility between the pre-COVID and post-COVID periods, as well as between France and the United States.

-The second part examines the leverage effect using TGARCH and EGARCH models.

-The third part provides an analysis and develops a predictive model for high-frequency data using Functional Principal Component Analysis.
