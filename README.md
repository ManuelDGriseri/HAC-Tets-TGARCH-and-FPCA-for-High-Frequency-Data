# HAC-tets-TGARCH-and-FPCA-for-high-frequency-data
\begin{itemize}
    \item \textbf{Part I:} Conducting appropriate tests for autocorrelation on the difference in volatility between the pre-COVID and post-COVID periods, as well as between  France and the United States.
    
    \item \textbf{Part II:} Analysis of the leverage effect using TGARCH and EGARCH models.
    
    \item \textbf{Part III:} Analysis and development of a predictive model for high-frequency data using Functional Principal Component Analysis (Functional PCA).
\end{itemize}
